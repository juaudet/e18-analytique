Instructions pour l'insertion des publicités

Il y a deux façons d'ajouter l'analytique à votre site web: via notre script ou notre API.

Script

Affichage des publicités

Mettre ce script dans votre balise <head>

(function(c,a,s,h){_var=s;_dom=c;_tok=h;document.write('<script src="'+c+'/'+a+'"><\/script>');})('https://e18-analytique-backend.herokuapp.com','analytique-gti525.js','gti','YOUR_TOKEN_HERE');
Ce script crée une variable JS `gti`. Si ce nom ne vous convient pas, vous pouvez la renommer en définissant un nouveau nom de variable dans le script ci-haut (dernier paramètre passé à la fonction);

Créer des containers vides pour les publicités.

Ceux-ci doivent posséder un ID afin d'être référencées au chargement des publicités.
<div id="banniere-1"></div>

Créer des styles pour définir les dimensions du container.

Cette étape est optionnelle mais fortement recommandée pour éviter le déplacement des éléments de la page lorsque la bannière est chargée.
/* CSS */
#banniere-1 {
width: 728px;
height: 90px;
}

Ajouter ce script juste avant la fermeture du <body>

*Ou dans une fonction prédéfinie

Exemple dans le <body>
<script>gti.banniere("#banniere-1", "horizontal");</style>


Exemple dans jQuery:
$(function () {
gti.banniere("#banniere-1", "horizontal");
}); Syntaxe: gti.banniere(selecteurCSS, format)

Formats acceptés
  • horizontal: 728 x 90
  • vertical: 120 x 600
  • mobile: 320 x 100

Envoi des infos de navigation

Pour envoyer les infos de navigation, ajoutez ce code de la même façon que pour la bannière:

gti.page("url");

Remplacez "url" par l'URL de la page affichée.

Voyez vos statistiques ici: https://e18-analytique-frontend.herokuapp.com/index.html#/admin/statistiques

API

Si votre page web ne se carge qu'une seule fois, comme c'est le cas pour les Single Page Application, vous pouvez utiliser directement notre API.

URL de base de l'API: https://e18-analytique-backend.herokuapp.com/api

Obtenir la bannière

GET /banniere

Paramètres:

  • format: le format de la publicité (horizontal, vertical ou mobile)
  • token: votre jeton d'API que vous trouvez à la page https://e18-analytique-frontend.herokuapp.com/index.html#/admin/generateur-code
  • struct: la structure des données retournées. Par défaut retourne un HTML, vous pouvez spécifier json si vous désirez plus de contrôle sur l'affichage.

Structure de la donnée retournée:

  • url: l'URL du lien de la publicité
  • image: l'image de la publicité
  • format: le format de la publicité (horizontal, vertical ou mobile)

Enregistrer une page vue

POST /page

Body:

  • url: l'URL de la page visitée
  • token: votre jeton d'API que vous trouvez à la page https://e18-analytique-frontend.herokuapp.com/index.html#/admin/generateur-code
Important: vous devez envoyer le cookie avec votre requête.
