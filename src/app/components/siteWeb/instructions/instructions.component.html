Instructions pour afficher des publicités dans les sites web

Cette operation se fait en plusieurs etapes:
  1. Mettre ce script dans le head
  2. Créer des containers vides pour les publicités. Ceux-ci doivent posséder un ID afin d'être référencées au chargement des publicités.
  3. Créer des styles pour définir les dimensions du container en attendant que la bannière soit chargée. Cette étape est optionnelle mais fortement recommandée pour éviter le déplacement des éléments de la page lorsque la bannière est chargée.
  4. Ajouter ce script juste avant la fermeture du body`, ou dans toute fonction exécutée lorsque le DOM est prêt.
